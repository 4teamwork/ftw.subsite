ftw.subsite
===========

``ftw.subsite`` provides a ``Subsite`` content type for embedding another website
within a Plone site.
It does this by defining it as navigation root.



Usage
-----

- Add ``ftw.subsite`` to your buildout configuration:

::

    [instance]
    eggs +=
        ftw.subsite

- Install the generic import profile.
